## App name
Hazi

## Short description
Trusted auctions, protected payments, and easier decluttering in Nigeria.

## Full description
Hazi helps people sell items they no longer need and discover trusted second-hand finds through transparent auctions.

Create listings, place visible bids, save auctions, and coordinate fulfilment from one account. Eligible transactions use Hazi wallet escrow so buyer funds remain protected until receipt and release conditions are met. Sellers can manage listings and delivery, buyers can track bids and orders, and authorised Hazi agents can manage assigned client sales.

Key features:
- Browse and search live auctions
- Create and manage item listings
- Place bids and follow auction activity
- Use protected wallet and escrow workflows
- Request delivery estimates and track booked delivery
- Submit verification, payment proof, and dispute evidence
- Receive account, bid, escrow, and delivery notifications
- Access dedicated buyer, seller, and agent workflows

Hazi changes how marketplace activity looks and flows without changing what bids, auctions, escrow releases, wallet transactions, agent assignments, or user roles mean.

## Required Play Console declarations
- Location: used only when the user confirms pickup coordinates or delivery routing.
- Photos/media: used when the user chooses listing, verification, payment-proof, or dispute-evidence images.
- Notifications: used for opted-in bid, escrow, delivery, and account updates.
- Financial features: wallet and escrow facilitate marketplace transactions; Hazi is not presented as a bank.
